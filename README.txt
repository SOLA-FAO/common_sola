Common repository for the Solutions for Open Land Administration
(SOLA) open source project. This repository contains projects common
across all SOLA products including;

   Common Utilities   - Contains utility classes, shared constants, 
                        exception classes.
   Common Services    - Contains classes common to the SOLA Service
                        layer including repository classes and
                        abstract classes for web services and EJBs.
   Common EJB Cache   - Singleton EJB used to cache service layer
                        entities.
   Common SWING       - Classes supporting the SOLA SWING client 
                        applications.
   Common Geotools UI - Extensions and enhancements to the Geotools
                        UI classes. 
   Common Messaging   - Contains the MessageUtility and messages for
                        the sola-common projects.
